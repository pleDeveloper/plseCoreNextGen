You are executing Wave 8 (Agentic Loop Wiring) for Pulse Core Next. Single-lane session on branch `wave8-agentic-loop` off `main`.

Closes the full AI-driven loop the product promises: a user emails the prospect (or a Recall bot joins a call), the transcript lands as `Conversation_Turn__c` rows, an agent automatically wakes up, reads the conversation, and proposes the next action (draft a reply email asking for missing info, or advance the workflow). The HITL approver sees the proposal in the Action Hub, approves, and the action executes.

# Scope details

## 1. ConversationTurnAgentTrigger + Queueable

Path: `packages/pulse-core/force-app/main/default/triggers/ConversationTurnAgentTrigger.trigger`

```apex
trigger ConversationTurnAgentTrigger on Conversation_Turn__c (after insert) {
    // Collect conversation ids → find their linked workflow instances.
    Set<Id> conversationIds = new Set<Id>();
    for (Conversation_Turn__c t : Trigger.new) {
        if (t.Conversation__c != null) conversationIds.add(t.Conversation__c);
    }
    if (conversationIds.isEmpty()) return;

    Map<Id, Id> convToInstance = new Map<Id, Id>();
    for (Conversation__c c : [
        SELECT Id, Workflow_Instance__c FROM Conversation__c
        WHERE Id IN :conversationIds AND Workflow_Instance__c != null
    ]) {
        convToInstance.put(c.Id, c.Workflow_Instance__c);
    }
    if (convToInstance.isEmpty()) return;

    Set<Id> instanceIds = new Set<Id>(convToInstance.values());

    // Only wake the agent for instances whose current state has a toolConfig
    // (i.e., the state is AI-driven, not human-only).
    Set<Id> aiEligibleInstanceIds = PulseAgentInvoker.filterAiDrivenInstances(instanceIds);
    if (aiEligibleInstanceIds.isEmpty()) return;

    for (Id instId : aiEligibleInstanceIds) {
        System.enqueueJob(new PulseAgentInvokerQueueable(instId));
    }
}
```

`PulseAgentInvoker` is a new helper class (`packages/pulse-core/force-app/main/default/classes/PulseAgentInvoker.cls`) with:

```apex
public with sharing class PulseAgentInvoker {
    @TestVisible private static Boolean enqueueSuppressed = false;

    public static Set<Id> filterAiDrivenInstances(Set<Id> instanceIds) {
        // Load each instance's pinned contract, parse, look up the State for
        // Current_State__c, return only ids whose state has a non-null
        // toolConfig with a non-blank toolKey.
    }
}

public with sharing class PulseAgentInvokerQueueable implements Queueable, Database.AllowsCallouts {
    private Id instanceId;
    public PulseAgentInvokerQueueable(Id instanceId) { this.instanceId = instanceId; }
    public void execute(QueueableContext ctx) {
        PulseAgentRunner.RunResult result = PulseAgentRunner.run(instanceId);
        // Log via Workflow_Event__c for observability. Do not throw — the
        // queueable's job is to best-effort attempt the agent; failures
        // surface in the returned RunResult and the event log.
    }
}
```

Trigger must be bulk-safe — the `filterAiDrivenInstances` call + enqueue loop both scale to 200 records per trigger invocation without hitting governor limits (parse the contract once per unique definition id, not per instance).

## 2. PulseInboundEmailHandler

Path: `packages/pulse-core/force-app/main/default/classes/PulseInboundEmailHandler.cls`

```apex
global with sharing class PulseInboundEmailHandler implements Messaging.InboundEmailHandler {

    global Messaging.InboundEmailResult handleInboundEmail(
        Messaging.InboundEmail email, Messaging.InboundEnvelope env
    ) {
        Messaging.InboundEmailResult result = new Messaging.InboundEmailResult();

        // Routing: if the subject contains "[Pulse:<instanceId>]", use that.
        // Otherwise try to match email.fromAddress to a Contact whose Opportunity
        // has an active Workflow_Instance__c. If both fail, still ingest but
        // leave Workflow_Instance__c = null (will route to the Conversation Hub
        // unattached).

        Id instanceId = PulseEmailRouter.extractInstanceId(email.subject);
        if (instanceId == null) {
            instanceId = PulseEmailRouter.routeByContactEmail(email.fromAddress);
        }

        // Build a ConversationInput (see ConversationIngestionService for shape)
        ConversationIngestionService.ConversationInput input =
            PulseEmailRouter.buildConversationInput(email, instanceId);

        ConversationIngestionService.ingest(
            new List<ConversationIngestionService.ConversationInput>{ input }
        );

        result.success = true;
        return result;
    }
}
```

`PulseEmailRouter` helper:
- `extractInstanceId(String subject)` — regex match `\[Pulse:(a0[0-9a-zA-Z]{13,15})\]` — returns the captured Id or null.
- `routeByContactEmail(String fromAddress)` — SOQL: find Contact with Email = fromAddress, join to OpportunityContactRole, join to Workflow_Instance__c where Parent_Record_Id__c = Opportunity.Id AND Status = 'Active'. Return the most recent match or null.
- `buildConversationInput(email, instanceId)` — normalizes the email into ConversationInput (set medium='Email', source_adapter='inbound_email', external_source_id=email.messageId, parent=the resolved instance's parent if known). Turn shape: role='Customer', speaker=email.fromName || email.fromAddress, content=email.plainTextBody (or htmlBody stripped).
